# The romanization table (https://www.loc.gov/catdir/cpso/romanization/amharic.pdf)
# gives no indication on how to distinguish the cases for the 6th order 
# For example: ህ he/h, ል le/l, ም me/m 
# ScriptShifter always maps to the consonant + vowel case.
# It seems like the only thing we can do better without language knowledge 
# is to handle when they occur at the end of words (map to consonant only).

# https://en.wikipedia.org/wiki/Ge%CA%BDez_script#Ge%CA%BDez_abugida
# "To represent a consonant with no following phonemic vowel, for example 
# at the end of a syllable or in a consonant cluster, the ə (/ɨ/) form is 
# used (the character in the sixth column)."

# https://en.wikipedia.org/wiki/Help:IPA/Amharic
# " Although the Amharic script can form simple syllables with one letter, 
# it may take multiple letters to form one complex syllable. The complex 
# syllables are formed using the sixth order, which serves the purposes 
# of being a vowel carrier and of marking a mute consonant (without a vowel). 
# For example, the Amharic word for "name", ስም, is one syllable but uses two 
# letters. Although it is pronounced as /sɨm/ it could also be read as /ˈsɨmɨ/. 
# *There is no way to distinguish the sixth order's functions in written Amharic.*"

# https://learnamharic.wixsite.com/learnamharic/amharic-script-1
# "The sixth order is always pronounced as the consonant alone when at the end of the word."

# There is a built-in transform "Ethiopic-Latin/ALALOC" in ICU 
# but it outputs e.g. c/ce for 'ች' instead of 'če/č'
